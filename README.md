Polymer Project
========

Web Dashboard built in Polymer

### Getting Started ###
* `npm install`
* `bower install`
* `npm install -g gulp`
* `gulp sass` (only if .css files do not exists)

# run
* `cd demo-server`
* `node app.js`

# user
* http://localhost:8080/ Web App
* http://localhost:5000/ Json Server
* http://localhost:8080/tests/ Testing Components
